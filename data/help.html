This extension is designed to have a simple workflow for choosing papers for Hot Topics during the Hotter Topics meetings.

  1. (First Time) Register at WordPress and obtain permission to post to the Hot Topics blog.
  2. (First Time) Go into this extension's preferences, and set a directory to save the articles posted.
  3. From the flame icon in the toolbar, the first link will open a titles-only view of Astro-ph.
  4. During Hotter Topics, open new tabs for the arXiv articles of interest.
  5. From the flame icon in the toolbar, choose the option to save the articles. Select categories for each article, then hit "save" at the button of the panel.
  6. A list will be created in your chosen folder. Click the notification that briefly pops up to easily open it.
  7. Email (or post) to the WordPress blog.

Please let me know of any problems/bugs, feature requests, suggestions for improvement, etc. The source code is available on GitHub.
